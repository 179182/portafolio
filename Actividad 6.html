APIs web en acción

API Web Storage

La API Web Storage proporciona un modo sencillo de almacenar datos en el navegador del usuario de manera persistente. Esta API incluye dos objetos: localStorage y sessionStorage. Estos objetos pueden almacenar pares clave-valor en el navegador del usuario.
